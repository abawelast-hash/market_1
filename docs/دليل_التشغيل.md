## 📋 المتطلبات الأساسية (ماذا تحتاج قبل البدء)

| المتطلب | الوصف | كيف تحصل عليه |
|---------|-------|---------------|
| **Node.js** | بيئة تشغيل JavaScript | حمّله من [nodejs.org](https://nodejs.org) |
| **npm** | مدير المكتبات | يأتي مع Node.js تلقائياً |
| **Git** | نظام التحكم بالإصدارات | حمّله من [git-scm.com](https://git-scm.com) |

### التحقق من التثبيت:
```bash
node --version    # يجب أن يظهر رقم مثل: v25.7.0
npm --version     # يجب أن يظهر رقم مثل: 10.x.x
git --version     # يجب أن يظهر رقم مثل: 2.x.x
```

## 🏁 التشغيل لأول مرة

### الخطوة 1: تنزيل المشروع
```bash
git clone https://github.com/abawelast-hash/market_1.git
cd market_1
```

### الخطوة 2: تثبيت المكتبات
```bash
npm install
```
> هذا الأمر يقرأ `package.json` ويُنزّل كل المكتبات المطلوبة في مجلد `node_modules`

### الخطوة 3: تعبئة قاعدة البيانات
```bash
npm run seed
```
> أو:
```bash
node database/seed.js
```
> هذا يُنشئ ملف `database/dalil.db` ويملأه بالبيانات التجريبية (متاجر، منتجات، تصنيفات)

### الخطوة 4: تشغيل الخادم
```bash
npm start
```
> أو:
```bash
node server.js
```

### الخطوة 5: فتح التطبيق
- افتح المتصفح على: **http://localhost:3000**
- على الهاتف (نفس الشبكة): **http://[IP-الكمبيوتر]:3000**

## 🔧 الأوامر المتاحة

| الأمر | الوظيفة | متى تستخدمه |
|-------|---------|------------|
| `npm start` | تشغيل الخادم | كل مرة تريد تشغيل التطبيق |
| `npm run dev` | تشغيل بوضع التطوير | أثناء التعديل على الكود |
| `npm run seed` | تعبئة البيانات | أول مرة أو لإعادة ضبط البيانات |
| `npm run setup` | تثبيت + تعبئة | أول مرة فقط |
| `npm run docs:watch` | تشغيل جاسوس التوثيق | لتحديث التوثيق تلقائياً |

## 📱 كيف تثبّت التطبيق كـ PWA على الهاتف

1. افتح الموقع في Safari
2. اضغط زر المشاركة (المربع مع السهم ↑)

4. اضغط **"إضافة"**

## 🗄️ إدارة قاعدة البيانات

### موقع قاعدة البيانات:
```
market_1/database/dalil.db
```

### إعادة ضبط البيانات:
```bash
# حذف قاعدة البيانات القديمة
del database\dalil.db    # على Windows
rm database/dalil.db     # على Mac/Linux

# إعادة التعبئة
npm run seed
```

### الحفظ الاحتياطي:
```bash
# نسخ قاعدة البيانات
copy database\dalil.db database\dalil_backup.db    # Windows
cp database/dalil.db database/dalil_backup.db      # Mac/Linux
```

### ملاحظة مهمة:
- قاعدة البيانات تُحفظ تلقائياً كل **30 ثانية**
- تُحفظ أيضاً عند إيقاف الخادم (Ctrl+C)
- لا تحذف الملف أثناء عمل الخادم!

## 🌐 النشر على الإنترنت (Deployment)

### الخيار 1: خادم VPS (مثل DigitalOcean, Vultr)

```bash
# 1. الاتصال بالخادم
ssh user@your-server-ip

# 2. تنزيل المشروع
git clone https://github.com/abawelast-hash/market_1.git
cd market_1

# 3. التثبيت والتشغيل
npm install --production
npm run seed
npm start
```

### الخيار 2: Railway / Render (مجاني)

1. اربط حسابك بـ GitHub
2. اختر مستودع `market_1`
3. المنصة ستكتشف Node.js تلقائياً
4. اضغط **Deploy**

### الخيار 3: استضافة محلية

```bash
# تشغيل على الشبكة المحلية
node server.js
# الآن أي جهاز على نفس الشبكة يقدر يفتح:
# http://[IP-جهازك]:3000
```

## 🔍 مراقبة التطبيق

### سجلات الخادم (Logs):
- الخادم يطبع رسائل في Terminal عند البدء
- أي خطأ يظهر باللون الأحمر

### أدوات المتصفح (DevTools):
1. اضغط **F12** في المتصفح
2. تبويب **Console**: رسائل JavaScript
3. تبويب **Network**: طلبات API
4. تبويب **Application**: 
   - Service Workers
   - IndexedDB
   - Cache Storage
   - Manifest

### التحقق من Service Worker:
1. F12 → Application → Service Workers
2. يجب أن يظهر `sw.js` بحالة **"Activated and running"**

### التحقق من قاعدة البيانات المحلية:
1. F12 → Application → IndexedDB
2. ابحث عن `dalil-raqqa-db`
3. يجب أن ترى المخازن: stores, products, categories, wishlist

## 🐛 حل المشاكل

### المشكلة: "Cannot find module 'express'"
```bash
# الحل: أعد تثبيت المكتبات
npm install
```

### المشكلة: "EADDRINUSE: port 3000"
```bash
# الحل 1: أوقف العملية القديمة
# Windows:
taskkill /f /im node.exe

# الحل 2: استخدم منفذ آخر
set PORT=8080 && node server.js    # Windows
PORT=8080 node server.js           # Mac/Linux
```

### المشكلة: "الصفحة فارغة"
1. افتح F12 → Console
2. ابحث عن رسائل الخطأ
3. تأكد أن الخادم يعمل
4. تأكد أن قاعدة البيانات موجودة

### المشكلة: "التطبيق لا يعمل أوفلاين"
1. افتح التطبيق وأنت متصل بالإنترنت
2. انتظر حتى يُخزّن الملفات (أول مرة)
3. بعدها أطفئ الإنترنت وجرّب

### المشكلة: "خطأ في قاعدة البيانات"
```bash
# احذف القاعدة وأعد إنشاءها
del database\dalil.db
node database/seed.js
node server.js
```

## 📂 هيكل المشروع للتعديل

### إذا تريد تعديل الألوان:
- الملف: `public/css/style.css`
- ابحث عن `:root` في أول الملف
- غيّر قيم الألوان

### إذا تريد تعديل المتاجر والمنتجات:
- الملف: `database/seed.js`
- عدّل البيانات
- أعد تشغيل: `npm run seed`

### إذا تريد إضافة صفحة جديدة:
1. أضف الدالة في `public/js/pages.js`
2. أضف المسار في `public/js/app.js` (في `setupRoutes`)
3. أضف الرابط في `public/index.html` (القائمة)

### إذا تريد إضافة API جديد:
1. أنشئ ملف في `routes/`
2. اربطه في `server.js`
3. أضف دالة في `public/js/api.js`

## 🔄 تحديث التطبيق

### عند تعديل الكود:
1. أوقف الخادم (Ctrl+C)
2. عدّل الملفات
3. أعد تشغيل: `npm start`

### عند تحديث Service Worker:
1. غيّر رقم الإصدار في `public/sw.js` (مثلاً `dalil-raqqa-v2`)
2. المتصفح سيكتشف التحديث ويُنزّل النسخة الجديدة
3. سيظهر إشعار "تم تحديث التطبيق" للمستخدم

### عند إضافة ملف ثابت جديد:
- أضفه في مصفوفة `STATIC_ASSETS` في `public/sw.js`
