## 🎯 ما هو هذا التطبيق؟ (شرح مبسّط)

تخيّل أنك تريد شراء غسالة أو ثلاجة أو أثاث لبيتك في مدينة الرقة.  
عادةً ستضطر تروح من مستودع لمستودع وتسأل عن الأسعار وتتعب.

هذا التطبيق مثل **"دليل الهاتف" لكن للمتاجر والمستودعات**:
- تفتحه على هاتفك 📱
- تشوف كل المتاجر والمستودعات في الرقة 🏪
- تقارن الأسعار بين المتاجر 💰
- تحفظ المنتجات اللي عجبتك بقائمة أمنيات ❤️
- تشارك المنتجات على واتساب مع أهلك 📲
- **يشتغل بدون إنترنت!** 📡
- تكسب نقاط ولاء (Market Coins) 🪙
- ضمان رقمي (Escrow) لحماية الشراء 🔐

## 🏗️ كيف يعمل التطبيق؟

### الفكرة البسيطة:

```
المستخدم (هاتف) ←→ الخادم (سيرفر) ←→ قاعدة البيانات (مخزن المعلومات)
```

### الشرح المفصّل:

#### 1️⃣ المستخدم يفتح التطبيق
- يكتب عنوان الموقع في المتصفح أو يضغط على أيقونة التطبيق
- التطبيق يُحمّل من الخادم (السيرفر)
- بعد أول مرة، التطبيق يتخزن على الهاتف ويشتغل بدون إنترنت

#### 2️⃣ التطبيق يطلب البيانات
- لما المستخدم يفتح صفحة المتاجر، التطبيق يرسل طلب للخادم
- الخادم يروح لقاعدة البيانات يجيب المعلومات
- يرجعها للتطبيق بشكل منظم (JSON)

#### 3️⃣ التطبيق يعرض البيانات
- يأخذ البيانات ويحولها لبطاقات جميلة
- كل منتج ببطاقة فيها: الاسم، السعر، المتجر، صورة

#### 4️⃣ العمل بدون إنترنت (Offline-First)
- التطبيق يحفظ نسخة من البيانات على هاتف المستخدم
- إذا انقطع الإنترنت، يعرض البيانات المحفوظة
- العمليات المعلقة تُخزّن في `sync_queue` وتُرسل عند عودة الاتصال

## 🖥️ الأنظمة والتقنيات

### البنية التقنية:

```
┌─────────────────────────────────────────────┐
│           المتصفح (Frontend) - 10 ملفات JS  │
│  HTML + CSS + JavaScript (Vanilla)          │
│  PWA + Service Worker + IndexedDB           │
├─────────────────────────────────────────────┤
│      الخادم (Backend) - 7 ملفات مسارات      │
│  Node.js + Express.js (24 endpoint)         │
├─────────────────────────────────────────────┤
│        قاعدة البيانات - 14 جدول             │
│  SQLite عبر sql.js                          │
├─────────────────────────────────────────────┤
│     مولّدات البيانات (Simulation) - V1+V2   │
│  Python 3.14 + Faker + pandas + tqdm        │
└─────────────────────────────────────────────┘
```

### شرح كل تقنية:

| التقنية | ما هي؟ | لماذا نستخدمها؟ |
|---------|--------|----------------|
| **Node.js** | بيئة تشغيل JavaScript على الخادم | سريع وبسيط ومجاني |
| **Express.js** | إطار عمل لبناء واجهات API | يسهّل إنشاء مسارات الخادم |
| **sql.js** | قاعدة بيانات SQLite تعمل بـ JavaScript فقط | لا تحتاج تثبيت برامج إضافية |
| **Python 3** | لغة برمجة لتوليد بيانات المحاكاة | مكتبات ذكاء اصطناعي وبيانات |
| **Faker** | مكتبة توليد بيانات وهمية واقعية | أسماء عربية، عناوين، أرقام |
| **PWA** | تطبيق ويب تقدمي | يتحول لتطبيق على الهاتف |
| **Service Worker** | برنامج يعمل في الخلفية | يخزّن الملفات للعمل أوفلاين |
| **IndexedDB** | قاعدة بيانات داخل المتصفح | تخزين محلي على جهاز المستخدم |

## 🔐 نظام الضمان الرقمي (Escrow)

```
المشتري → POST /api/qr/transaction → رمز QR فريد
                ↓
البائع ← GET /api/qr/verify/:txCode ← تحقق
                ↓
البائع → POST /api/qr/release/:txCode → إفراج + 20 نقطة للمشتري
```
