## 📄 server.js - قلب الخادم

**الموقع:** `market_1/server.js`  
**الوظيفة:** نقطة البداية - يُشغّل الخادم ويربط كل الأجزاء

### ماذا يفعل بالتفصيل:
1. **يستورد المكتبات** (Express, CORS, Compression)
2. **يُنشئ المجلدات المطلوبة** (database, uploads) إذا لم تكن موجودة
3. **يضبط الإعدادات:**
   - تفعيل ضغط البيانات (Compression)
   - تفعيل CORS
   - تفعيل استقبال JSON
4. **يخدم الملفات الثابتة** من مجلد `public`
5. **يربط مسارات API** (المتاجر، المنتجات، التصنيفات، البحث، QR، الإحصائيات)
6. **يخدم `index.html`** لأي مسار غير API (SPA fallback)
7. **يتعامل مع الأخطاء** ويعرض رسالة خطأ عربية
8. **يبدأ الاستماع** على المنفذ 3000

### كيف يتصل بالملفات الأخرى:
```
server.js
  ├── يستورد routes/stores.js      ← /api/stores
  ├── يستورد routes/products.js    ← /api/products
  ├── يستورد routes/categories.js  ← /api/categories
  ├── يستورد routes/qr.js          ← /api/qr
  ├── يستورد routes/search.js      ← /api/search
  ├── يستورد routes/stats.js       ← /api/stats
  └── يخدم public/                  ← الملفات الثابتة
```

## 📁 database/schema.js - مهندس قاعدة البيانات

**الموقع:** `market_1/database/schema.js`  
**الوظيفة:** إنشاء هيكل قاعدة البيانات + وظائف مساعدة للاستعلام

### الوظائف المُصدّرة:

| الوظيفة | المعاملات | الوصف |
|---------|----------|-------|
| `getDb()` | - | جلب اتصال قاعدة البيانات (async) |
| `saveDb()` | - | حفظ قاعدة البيانات على القرص |
| `allRows(sql, params)` | استعلام SQL + معاملات | تنفيذ استعلام وإرجاع كل السطور كمصفوفة |
| `getRow(sql, params)` | استعلام SQL + معاملات | تنفيذ استعلام وإرجاع سطر واحد |
| `run(sql, params)` | استعلام SQL + معاملات | تنفيذ أمر (إدراج/تحديث/حذف) |

### الجداول التي يُنشئها:
1. `categories` - التصنيفات
2. `stores` - المتاجر
3. `products` - المنتجات
4. `product_images` - صور المنتجات
5. `reviews` - التقييمات
6. `wishlists` - قوائم الأمنيات
7. `qr_scans` - سجل مسح QR
8. `shares` - سجل المشاركات
9. `category_filters` - فلاتر التصنيفات

### الفهارس (Indexes) - لتسريع البحث:
- `idx_products_store` - البحث في منتجات متجر معين
- `idx_products_category` - البحث في منتجات تصنيف معين
- `idx_products_price` - ترتيب حسب السعر
- `idx_products_featured` - فلترة المنتجات المميزة
- `idx_stores_neighborhood` - البحث حسب الحي
- `idx_stores_featured` - فلترة المتاجر المميزة
- `idx_wishlists_device` - قائمة أمنيات جهاز معين
- `idx_product_images_product` - صور منتج معين

## 📁 routes/ - مسارات API

### routes/stores.js - مسارات المتاجر

| المسار | الطريقة | الوظيفة | المعاملات |
|--------|---------|---------|----------|
| `/api/stores` | GET | قائمة المتاجر | `neighborhood`, `featured`, `page`, `limit`, `sort` |
| `/api/stores/neighborhoods` | GET | قائمة الأحياء مع عدد المتاجر | - |
| `/api/stores/:slug` | GET | تفاصيل متجر واحد | `slug` في الرابط |

**ماذا يرجع عند طلب تفاصيل متجر:**
- معلومات المتجر كاملة

- تصنيفات منتجاته

### routes/products.js - مسارات المنتجات

| المسار | الطريقة | الوظيفة | المعاملات |
|--------|---------|---------|----------|
| `/api/products` | GET | قائمة المنتجات | `category`, `store`, `featured`, `solar`, `min_price`, `max_price`, `brand`, `sort`, `page` |

| `/api/products/:id/share` | POST | تسجيل مشاركة | `device_id`, `platform` في الجسم |
| `/api/products/wishlist/:device_id` | GET | قائمة أمنيات جهاز | `device_id` في الرابط |

**خيارات الترتيب المتاحة:**
- `featured` - الأكثر تميزاً (افتراضي)
- `price_asc` - السعر من الأقل
- `price_desc` - السعر من الأعلى
- `newest` - الأحدث
- `popular` - الأكثر مشاهدة
- `wishlist` - الأكثر رغبة

## 📁 public/ - ملفات الواجهة

### public/index.html - الهيكل العظمي

**المكونات:**
1. **الهيدر (Header):** زر القائمة + الشعار + أزرار البحث والأمنيات
2. **شريط البحث:** حقل إدخال + اقتراحات تلقائية

4. **منطقة المحتوى (Main Content):** يتغير محتواها حسب الصفحة

### public/sw.js - Service Worker

**استراتيجيات التخزين المؤقت:**

| نوع الطلب | الاستراتيجية | الشرح |
|-----------|-------------|-------|
| طلبات API | Network-first | يحاول الإنترنت أولاً، إذا فشل يرجع للمخزّن |
| الصور | Cache-first | يبحث في المخزّن أولاً (الصور لا تتغير كثيراً) |
| ملفات CSS/JS | Cache-first | تُخزّن عند أول زيارة |
| صفحات HTML | Network-first | يحاول جلب أحدث نسخة |
