## 📄 server.js — قلب الخادم (72 سطر)

**الوظيفة:** نقطة البداية — يُشغّل الخادم ويربط كل الأجزاء

### ماذا يفعل:
1. **يستورد المكتبات** (Express, CORS, Compression)
2. **يُنشئ المجلدات** (database, uploads) إذا لم تكن موجودة
3. **يضبط الإعدادات:** Compression + CORS + JSON parsing
4. **يخدم الملفات الثابتة** من `public/` (مع cache headers)
5. **يربط 7 مسارات API:**
   ```
   /api/stores      ← routes/stores.js
   /api/products    ← routes/products.js
   /api/categories  ← routes/categories.js
   /api/qr          ← routes/qr.js (+ Escrow)
   /api/search      ← routes/search.js (+ Visual)
   /api/stats       ← routes/stats.js
   /api/loyalty     ← routes/loyalty.js (🪙 جديد)
   ```
6. **SPA fallback:** يخدم `index.html` لأي مسار غير API
7. **معالج أخطاء:** رسائل عربية + stack trace في dev mode
8. **يبدأ الاستماع** على المنفذ 3000
