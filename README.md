## 📖 عن المشروع

تطبيق **"دليل الرقة"** هو دليل رقمي شامل للمستودعات والأسواق المنزلية في مدينة الرقة.
يمكن للمستخدمين تصفح المتاجر والمنتجات، مقارنة الأسعار، مشاركة المنتجات عبر واتساب،
وحفظ المنتجات المفضلة - كل ذلك يعمل حتى بدون إنترنت!

### ✨ المزايا الرئيسية

- 🔍 **بحث شامل** - ابحث بالاسم أو العلامة التجارية أو اللون
- 📂 **10 تصنيفات** - أثاث، أجهزة كهربائية، سولار، وأكثر
- 🏪 **دليل متاجر** - معلومات التواصل والموقع لكل متجر
- 💰 **مقارنة أسعار** - بالليرة السورية والدولار
- ❤️ **قائمة أمنيات** - احفظ المنتجات لترجع لها لاحقاً
- 📲 **مشاركة واتساب** - شارك أي منتج أو متجر مباشرة
- 📡 **يعمل أوفلاين** - بدون إنترنت بعد أول زيارة
- 📱 **PWA** - قابل للتثبيت كتطبيق على الهاتف
- ☀️ **فلتر سولار** - منتجات متوافقة مع الطاقة الشمسية
- 📊 **رموز QR** - لكل متجر ومنتج

## 🚀 التشغيل السريع

```bash
# 1. استنساخ المشروع
git clone https://github.com/abawelast-hash/market_1.git
cd market_1

# 2. تثبيت المكتبات
npm install

# 3. تعبئة البيانات التجريبية
npm run seed

# 4. تشغيل الخادم
npm start

# 5. افتح المتصفح على
# http://localhost:3000
```

---

## 🛠️ التقنيات المستخدمة

| التقنية | الوصف |
|---------|-------|
| **Node.js** + **Express** | الخادم الخلفي |
| **sql.js** (SQLite) | قاعدة البيانات |
| **HTML/CSS/JS** (Vanilla) | الواجهة الأمامية |
| **Service Worker** | العمل أوفلاين |
| **IndexedDB** | تخزين محلي |
| **PWA** | تطبيق ويب تقدمي |

## 📋 الأوامر المتاحة

| الأمر | الوصف |
|-------|-------|
| `npm start` | تشغيل الخادم |
| `npm run seed` | تعبئة البيانات |
| `npm run setup` | تثبيت + تعبئة |
| `npm run docs:watch` | 🕵️ تشغيل جاسوس التوثيق |

---

## 🕵️ نظام الجاسوس (التوثيق الآلي)

المشروع يحتوي على نظام مراقبة ذكي يقوم بـ:

- ✅ رصد أي تغيير في ملفات المشروع (إنشاء/تعديل/حذف)
- ✅ تسجيل التغييرات في سجل مُفصّل بالعربي مع الوقت والتاريخ
- ✅ تحديث ملفات التوثيق تلقائياً
- ✅ عرض إشعارات ملونة في Terminal

```bash
# لتشغيل الجاسوس
npm run docs:watch
```

## 🌐 API Endpoints

| المسار | الطريقة | الوصف |
|--------|---------|-------|
